HOW TO ADD YOUR PHOTO
=====================

1) Save your photo as one of these filenames in this folder:
     savitha.jpg      (recommended)
     savitha.jpeg
     savitha.png

2) Square crops work best (the site shows it inside a circular frame).
   A size around 600×600 px is plenty.

3) Refresh the site — your photo will appear in the hero and in the
   top-left nav avatar automatically. No code changes needed.

Until you add the photo, the styled "SN" placeholder (savitha.svg) is
displayed.
